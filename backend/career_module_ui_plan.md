# Career Module UI/UX Blocks

## SchoolAdmin
- Управление тестами (создание, редактирование, публикация)
- Просмотр результатов учеников
- Аналитика по классам/школе (Radar Chart, распределение интересов)
- История прохождений (если разрешено)
- Кнопка «Назад»

## Student
- Прохождение теста (адаптивная форма, RU/UZ)
- Radar Chart (Chart.js, единый стиль)
- Текстовая интерпретация (топ-3 сферы)
- Рекомендации предметов
- Блок «Профориентация» в профиле (последний результат + дата)
- История (если разрешено)
- Кнопка «Назад»

## SuperAdmin
- Глобальная статистика (без редактирования)

## Общие требования
- Полная поддержка RU/UZ
- Адаптивность (desktop + mobile)
- RBAC: доступ только к своей школе/результатам
- Audit-лог действий

## Future-ready
- AI-анализ
- PDF-отчёт
- Сравнение интересов
- Рекомендации профессий
