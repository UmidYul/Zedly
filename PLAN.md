Проект ZEDLY — это веб\-сайт, состоящее из клиентской и серверной частей. Клиент реализован с использованием современных веб\-технологий (HTML, CSS, JavaScript), а серверная часть — на Node.js. Проект предназначен для создания интерактивной платформы с возможностью аутентификации пользователей, отображения различных страниц (например, landing page и личного кабинета), а также взаимодействия с сервером для получения и отправки данных. В основном это для легкого мониторинга прогресса для всех(для директоров, для учителей и учеников), а также с возможностями добавлять тесты для учителей и учеников, чтобы проверять их знания. Еще один из функцией для учеников это определение их интересов через тест на профориентацию и рекомендовать предметы по этим сферам.

## **Технологии и структура**

* Клиент: HTML, CSS (SCSS), JS (ES6+), i18next, адаптивная верстка.  
* Сервер: Node.js (Express), REST API  
* Аутентификация: JWT или сессии.  
* Хранение данных: Postgre SQL

1\. Безопасность 

1. RBAC \+ object-level checks

2. Rate limiting

3. JWT access \+ refresh (rotation)

4. Audit log всех критичных действий

5. Upload security (image only, size limit)  

2\. Роли

7. \- SuperAdmin (платформа)  
8. \- SchoolAdmin (школа)  
9. \- Teacher (школа)  
10. \- Student (школа)

12. \- Каждый API-запрос проверяет:  
13.   \- роль  
14.   \- принадлежность к школе  
15.   \- владение объектом (IDOR-защита)
